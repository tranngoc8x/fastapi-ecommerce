# Python Code Structure Standards [L2]

## Organization
- Domain-driven design
- Single responsibility
- Separation of concerns
- Logic/IO separation
- Composition over inheritance

## Module Structure
- Clear purpose
- Limited size
- Related functionality
- Proper imports
- Documentation

## Class Design
- Single responsibility
- Clear interfaces
- Minimal coupling
- Proper encapsulation
- Inheritance hierarchy

## Function Design
- Single purpose
- Limited parameters
- Clear return values
- Error handling
- Documentation

## Code Layout
- Logical grouping
- Consistent style
- Clear separation
- Proper spacing
- Organized imports
